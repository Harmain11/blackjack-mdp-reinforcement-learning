# Blackjack MDP Reinforcement Learning

## Overview  
This notebook demonstrates reinforcement learning and dynamic programming techniques applied to the Blackjack Markov Decision Process (MDP). It includes modeling the Blackjack environment, building transition models, applying value iteration and policy iteration, and comparing SARSA and Q-learning algorithms.

## Features  
- Implementation of a Blackjack MDP with dynamic programming methods  
- Value Iteration and Policy Iteration algorithms for policy optimization  
- SARSA and Q-learning algorithms for model-free reinforcement learning  
- Visualization of value functions and policies for Blackjack states  
- Use of Gymnasium environment for Blackjack simulation  

## Tech Stack  
- Python  
- Jupyter Notebook / Google Colab  
- NumPy  
- Pandas  
- Matplotlib  
- Gymnasium  
- PyTorch  

## How to Use  
1. Clone the repository or download the notebook file.  
2. Run the notebook in Jupyter or Google Colab environment.  
3. Follow the cells to load libraries, build the Blackjack model, and run algorithms.  
4. Analyze plots and results to understand the convergence and policy behavior.
